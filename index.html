what is Zsh?

The name zsh derives from the name of Zhong Shao, a teaching assistant at Princeton University. Paul Falstad regarded Shao's login, "zsh", as a good name for a shell.
~ wikipedia
  • 1990 by Paul Falstad while a student at Princeton

Zsh alternatives:

  • bash Bourne Again Shell
  • fish Friendly Interactive Shell
  • nu   New Shell
  • bash was Created in 1989 by Brian Fox for the GNU Project,[7] it is designed as a completely free software alternative for the Bourne shell, sh, and other proprietary Unix shells

  • bourne shell: 1979
  • thomson shell: 1971 (simple command interpreter)
  • zsh is infact: a combo of KornShell ksh and tenex c shell
  • bourne shell The Bourne shell (sh) is a foundational Unix command-line interpreter and scripting language developed by Stephen Bourne at Bell Labs in 1979.

  • fish shell: 2005 w/ substantial updates over the years
  • nu shell: 2019
